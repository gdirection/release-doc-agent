# Troubleshooting Login Issues

## Password Login Failures

Password login failures usually occur when a user enters invalid credentials or
when the account is locked after repeated failed attempts.

## SSO Login Failures

SSO login failures can be caused by incorrect identity provider metadata,
callback URL configuration problems, or expired certificate validity. Confirm
that the configured identity provider values match the enterprise SSO settings.

## Expired Login Sessions

Expired login sessions require the user to start a new authentication flow and
complete login again.
