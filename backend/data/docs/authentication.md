# Authentication Setup Guide

## Password Login

Password login lets users authenticate with an email address and password. Account
administrators can enforce password policies and session expiration settings.

## SSO Configuration

Enterprise workspaces can use SAML-based single sign-on with a centralized
identity provider. Administrators configure identity provider metadata, callback
URLs, and certificate information before enabling SSO for their organization.

## Session Management

Session management controls how long authenticated sessions remain active and
how users are asked to reauthenticate after a session expires.
